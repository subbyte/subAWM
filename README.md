# subAWM
Awesome Window Manager Configuration

Features:
- widgets (CPU temp, SSD I/O, WiFi signal, Volume, Battery)
- floating console (toggle with Mod4 + ~)

Awesome Versions:
- v4.0
- v3.5.9

Other programs I use:
- gvim
- zathura
- firefox w/ vimperator
- chromium w/ vimium
